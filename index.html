🔑 Token do GitHub

O token permite salvar e ler seus checklists diretamente no GitHub Gist. Ele fica salvo apenas no seu navegador.

Como gerar seu Token (2 minutos)
  1. Acesse github.com/settings/tokens/new
  2. Em Note, coloque: BM Checklist
  3. Em Expiration, escolha No expiration
  4. Marque apenas o escopo: gist
  5. Clique em Generate token e copie o token
  6. Cole aqui no campo acima
